## 1. Project Vision

### What We're Building

**AI Secretary** - A full-stack SaaS application that serves as an AI-powered personal secretary, capable of:
- Drafting professional communications
- Creating and managing task lists
- Maintaining conversation context
- Providing intelligent assistance

### Why This Project?

| Goal | Rationale |
|------|-----------|
| **Learn LLM Integration** | Practical experience with Claude API |
| **Build Full-Stack Skills** | Frontend + Backend + Database |
| **Create Portfolio Piece** | Demonstrable, deployable project |
| **Understand SaaS Architecture** | Real-world patterns and practices |
| **Zero Cost Learning** | Using free tiers throughout |

### Who Is This For?

**Primary Learner Profile:**
- Software developer with existing programming knowledge
- Experience with at least one programming language
- Interest in AI/LLM technology
- Desire to build portfolio-worthy projects
- Self-directed learner

## 3. Technology Decisions

### Stack Selection

| Component | Technology | Rationale |
|-----------|------------|-----------|
| **LLM** | Claude API | Modern, capable, good documentation |
| **Backend Runtime** | Node.js | JavaScript ecosystem, async support |
| **Backend Framework** | Express.js | Simple, widely understood |
| **Database** | Supabase | PostgreSQL + Auth + Free tier |
| **Frontend** | React | Industry standard, component-based |
| **Styling** | Tailwind CSS | Rapid development, consistent design |
| **Hosting** | Vercel | Free tier, easy deployment |

### Why These Choices?

1. **JavaScript Everywhere** - Same language front and back
2. **Free Tiers** - Zero cost to learn and deploy
3. **Industry Relevant** - Skills transfer to real jobs
4. **Good Documentation** - Self-learning friendly
5. **Portfolio Ready** - Modern, professional stack
