# Part 3: Web Frontend

React web application providing a modern chat interface for the AI Secretary.

## Features

- ?? Modern, responsive UI with Tailwind CSS
- ?? User authentication (login/register)
- ?? Real-time chat interface
- ?? Conversation management (create/delete/switch)
- ? Fast and optimized (Vite)
- ?? Production deployment ready (Vercel)

## Prerequisites

- Node.js 18+
- Part 2 Backend API deployed and running

## Technologies Used

- React 18
- React Router (navigation)
- Tailwind CSS (styling)
- Vite (build tool)
- Vercel (deployment)

## Setup Instructions

See the [Part 3 Guide](../docs/Part_3_Frontend_Guide.md) for complete step-by-step instructions.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Environment Variables

```bash
VITE_API_URL=http://localhost:3000  # Your backend API URL
```
